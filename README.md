## About application
Fast and simple application for generating and comparison hashes from files or text

__Supporting algorithms:__
- MD5
- SHA-1
- SHA-224
- SHA-256
- SHA-384
- SHA-512
- CRC-32

__Languages:__
- English
- Russian
- German
